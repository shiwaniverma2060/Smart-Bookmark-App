# Smart Bookmark App

## Overview

Smart Bookmark App is a simple web application that allows users to save and manage bookmarks.

## Technologies

* React
* JavaScript
* HTML
* CSS
* GitHub
* Vercel

## Challenges and Solutions

### GitHub Push Conflict

Problem: Repository already contained files when pushing code.
Solution: Used force push to overwrite repository content.

### Deployment Understanding

Problem: Confusion about how deployment updates work.
Solution: Connected GitHub repository to Vercel for automatic deployment.
